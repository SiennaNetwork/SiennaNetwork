# Wrapped Sienna (ERC20) Token

# Run locally

## Clone the repo

```
$ git clone https://github.com/hackbg/wrapped-sienna
$ cd wrapped-sienna
```

## Install dependencies

```
$ npm install
```

## Compile

```
$ npx truffle compile
```

## Test

```
$ npx truffle test
```

## Coverage report

```
$ npx truffle run coverage
```

## Deployment

```
$ npx truffle migrate --network <network>
```
